Creates a shortcode to add a jQuery-powered countdown timer.

== Description ==

**WP jQuery Countdown** gives you a shortcode to easily use Martin Conte Mac Donellâ€™s [jQuery Countdown script](https://github.com/Reflejo/jquery-countdown) in posts or pages.

== Installation ==

1. Download the **wp-jquery-plugin.zip** file to your local machine.
2. Either use the automatic plugin installer *(Plugins - Add New)* or unzip the file and upload the **wp-jquery-plugin** folder to your **/wp-content/plugins/** directory.
3. Activate the plugin through the Plugins menu
4. Insert the timer into pages or posts by inserting the shortcode **[wp_jquery_countdown endtime="xx:xx"]** where **xx:xx** is the time (minutes:seconds) that you want the timer to count down.
